    Mini App Status

    Mini Programs can have one of a number of statuses depending on where it is in the review/release life cycle. In rough order they are:

    Pending Review : The Mini App has been submitted for review by the platform. It has not yet been assigned to a reviewer.

    In Review : A reviewer has been assigned and will get back to the developer with changes if necessary or change the status to approved.

    Review Cancelled : The developer has decided to cancel the review of the Mini App. The reviewer no longer needs to look at this version.

    Requires Changes : The reviewer found issues that don't conform to the platform's guidelines and the developer will need to re-submit again once changes have been made. The status will go back into Pending Review once the developer submits again.

    Approved : The reviewer has approved the Mini App with a semantic version vX.X.X

    Released : The developer has clicked the deploy button and the application should be released with the latest semantic version showing.
